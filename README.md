# fastapi-html-frontend-demo

Simple FastAPI File Manager
A lightweight web application that demonstrates how to handle file uploads and retrieval using FastAPI for the backend and a simple HTML/JavaScript frontend

File Upload: Select a file from your computer and store it in a local server folder.

File Retrieval: Fetch and read the contents of an uploaded file directly in the browser.

Asynchronous Backend: Built with FastAPI for high performance.

Clean UI: Simple, no-refresh interface using the Fetch API.
